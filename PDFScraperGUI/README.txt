
Hello! This tool allows users to select a PDF lab report (e.g., wastewater treatment results), extract pH, EC, ORP, and Temperature values,
and save the extracted results into a formatted CSV file.

1. Run the `.exe` file.
2. Select a PDF file when prompted.
3. Choose where to save the resulting CSV.
4. A confirmation will appear, and the CSV will open automatically.
